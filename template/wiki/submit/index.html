Phliosophy
________________________________________________________________________________

While some on this site may be strongly opinionated on certain topics, this
site seeks to provide a platform for experenced Power-Users and Devs to share
their ideas, guides, practices, and experences openly. 

We are community driven, and constantly looking for new contributers.
If you have something you want to say, simply fork this project

    $ git clone https://github.com/173duprot/wiki

and add an article following our @/formatting-guide


Plaintext

The text is deliminated at 80 characters, meaning it supports almost all
terminals. This site has an RSS feed, and can be accessed through a git
repository.
